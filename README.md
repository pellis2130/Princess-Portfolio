# Princess Ellis Portfolio

## Overview

This project is a personal portfolio website created as part of my Software Development coursework at ECPI University. The website demonstrates my growing knowledge of HTML, CSS, and JavaScript while providing visitors with information about me, my technical skills, and projects I have completed throughout my program.

The portfolio continues to evolve throughout the course as new JavaScript concepts and web development techniques are introduced.

---

## Features

- Responsive webpage layout
- About Me section
- Skills list generated with JavaScript
- Featured content section
- Links to GitHub projects
- Dark Mode toggle
- Contact form
- Dynamic DOM manipulation
- Timed notification and confirmation messages
- Interactive JavaScript event handling

---

## JavaScript Concepts Demonstrated

This project includes examples of:

- Variables and arrays
- Loops
- Conditional statements
- Functions
- Event listeners
- DOM manipulation
- Dynamic element creation
- `querySelector()` and `getElementById()`
- `createElement()` and `appendChild()`
- `event.preventDefault()`
- `setTimeout()`
